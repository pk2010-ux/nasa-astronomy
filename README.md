# NASA ASTRONOMY PICTURE OF THE DAY - WEBSITE
A website where astronomy enthusiasts can explore the NASA APOD.
## Features
*  Today's NASA APOD picture and a brief about it
*  **Date Picker Option** - Allows you to pick a date and view that specific day's NASA APOD image
*  A beautiful Aurora-based theme
* Eye catching fonts- **Space Grotesk** and **Outfit** used
## Conclusion
This website was created using the guide [Give Your Website a Pulse](https://stardance.hackclub.com/missions/nasa-page/guide) by Stardance
